## 概念

* 函数调用自己
* 需要有基线条件和递归条件

## 基线条件和递归条件

> 每个递归函数都有两部分：基线条件和递归条件。

* 递归条件是指函数调用自己
* 基线条件是指函数不再调用自己，从而避免无限循环

```javascript
//倒计时
function countdown(i) {
  console.log(i);
  if (i <= 0) {
    return; //基线条件
  } else {
    countdown(i - 1);
    //递归条件;
  }
}
```
